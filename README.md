# panduan *command line*

## Operasi *directory* (*folder*)

### melihat isi *directory*

### membuat *directory*/*folder*

```
$ mkdir nama-folder
```

### menghapus *directory*

```
$ rmdir nama-folder
```

### melihat isi *directory*

```
$ ls
```

atau

```
$ ls nama-folder
```

### pindah lokasi *directory*

```
$ cd nama-folder
```

untuk pindah ke *parent directory* (folder atasnya)

```
$ cd ..
```
